기본 알고리즘 메뉴 선택

기본 알고리즘 v.1: 이미지 8등분 분할 및 객체 탐지

목표: 단일 이미지 내 다수의 객체 탐지를 위한 기본 알고리즘 구현 및 검증.

방법: 입력 이미지를 8등분으로 분할하여 각 영역에 사람이 있는지 여부를 판단한다. 각 영역의 사람 존재 유무를 판단하여 최대 8명까지의 사람 존재 여부를 확인한다.

기본 알고리즘 v.1 카메라 모드: 실시간 이미지 8등분 분할 및 객체 탐지

목표: 기본 알고리즘 v.1을 실시간 카메라 영상에 적용하여 5초 간격으로 이미지를 분석한다.

방법:

  • 웹캠을 통해 실시간으로 영상을 캡처한다.
  • 캡처된 영상을 8등분으로 분할한다.
  • 각 분할된 이미지를 Teachable Machine 모델로 입력하여 사람 존재 여부를 판단한다.
  • 각 분할된 이미지에서 사람이 탐지되면 카운트를 증가시켜 최종적으로 이미지 내 사람의 수를 계산한다.
  • 5초 간격으로 위 과정을 반복한다.

기본 알고리즘 v.2: 이미지 2등분 분할 및 객체 탐지

목표: 기본 알고리즘 v.1의 간소화 및 실시간 적용성 강화

방법: 입력 이미지를 2등분(좌우)하여 각 영역에 사람이 있는지 여부를 판단한다. 좌우 영역에 사람이 있는지 여부를 확인하여 사람 존재 여부를 판별한다.

기본 알고리즘 v.3: 이미지 병합 및 k-평균 군집화

목표: 기존 알고리즘의 한계 극복 및 정확한 테이블별 사람 존재 여부 판별
